# bspwm

## install
git clone https://github.com/corey-truscott/bspwm.git ~/.config/bspwm

run via a display manager or if you use xinit: `echo "exec bspwm" > ~/.xinitrc`

## dependencies
- bspwm
- sxhkd
- polybar
- dunst
- flameshot *(optional)*
- syncthing *(optional)*
- mullvad-cli *(optional)*
- bc *(optional)*
